# AI Agent Skills

This directory contains context and documentation optimized for AI agents, following the AgentSkills.io standard.

## Structure
Each subdirectory represents a specific domain capability ("Skill").

- `booking/`: Managing appointments and slots.
- `tenancy/`: Multi-tenant isolation rules.
- `authentication/`: User identity and access.

## Usage
AI agents should read the `SKILL.md` file in the relevant directory to understand constraints and usage patterns before modifying code.
